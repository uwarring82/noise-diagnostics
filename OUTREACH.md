# About this project — in plain language

Status: draft — 2026-06-12

Trapped ions are single charged atoms held still by electric fields and used as the
memory of some quantum computers. Stray electrical noise heats their motion and spoils
calculations. Labs usually summarise that noise as a single number, which can make two
very different problems look identical. This project gives a recipe for telling those
problems apart — so a lab can pick the fix that actually works — and a way to check that
the diagnosis is right before trusting it.

What's here now: the full method (the essay) and its glossary. Still to come: the
verified source list is started but not yet closed, and any analysis tools.
