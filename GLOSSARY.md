# Glossary

## Harbour vocabulary

Closed list (FR4). These terms have no external definition, so they are defined here.
Each heading is a stable anchor (`#harbour`, `#harbourmaster`, …) that visitor files
link to.

### Harbour
The Open-Science Harbour — the ecosystem of frameworks, essays, and interfaces this
work belongs to. It makes maps, not territory: no metaphysical claims, boundaries
stated as conditions. <a id="harbour"></a>

### Harbourmaster
The default coordinating stance; it maintains best-effort consistency across the work
and rules on cross-stance conflicts. <a id="harbourmaster"></a>

### Coastline
A framework or boundary condition imported as a constraint and referenced, never
restated. Fundamental physics (here: macroscopic QED, quantum mechanics, the
Lévy–Khintchine decomposition) enters as an external Coastline. <a id="coastline"></a>

### Sail
An interpretive, operational synthesis — an essay or protocol narrative — that imports
from Coastlines without itself being a fundamental law. This repository's essay is a
Sail. <a id="sail"></a>

### witness
A peripheral reference mark that lets a maintained point be relocated and checked —
triangulation rather than a single unverifiable claim. (Here: the pinned commit, asset
checksums, and verified identifiers are witnesses.) <a id="witness"></a>

### +EC
The collective / the error-correction discipline: corrections kept as a visible record
rather than silently absorbed. The logbook is the running +EC record. <a id="ec"></a>

## Terms used here

Domain vocabulary — **link, don't redefine**. Canonical definitions live in the
literature; point to the bibliography entry or the essay.

- **heating rate ($\dot{\bar n}$)** — see [BIBLIOGRAPHY.md](BIBLIOGRAPHY.md) (Brownnutt *et al.* 2015) and the essay §1.
- **Lévy–Khintchine decomposition** — the structure theorem for infinitely divisible
  processes underlying the three operational classes; see the essay §5 and the
  Holevo / Vacchini imports in [BIBLIOGRAPHY.md](BIBLIOGRAPHY.md).
- **translation-covariant Markov semigroup** — see [BIBLIOGRAPHY.md](BIBLIOGRAPHY.md)
  (Holevo 1996; Vacchini & Hornberger 2009).
- **event-count parameter ($\lambda\tau$)**, **Markov-accessibility parameter
  ($\mu = \tau/\tau_c$)**, **Decision Engine**, **three operational classes** — defined
  in the essay (§4–§7) and its Appendix A glossary.
